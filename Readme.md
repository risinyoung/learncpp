### 简介

这是一个学习 C++，Git 的试验项目，以杨道奇的 C++ 面向对象数值计算为纲，中间穿插其它C++书籍，比如 primer plus 里的内容。Git 的学习以廖雪峰的网页为主，学习 Git Bash 的使用。
